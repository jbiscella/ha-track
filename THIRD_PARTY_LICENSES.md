# Third-Party Licenses

This project is licensed under the [BSD Zero Clause License](LICENSE) (0BSD).
The dependency table below is generated by the build; do not edit it by hand.
Regenerate with:

    ./mvnw license:aggregate-add-third-party

The four core library modules — `commons`, `heerwisch-api`, `frau-holle` and
`nachtkrapp` — are JDK-only and pull in no third-party runtime code; their only
non-test dependencies are sibling 0BSD modules of this repository.

JFreeChart is the only LGPL-only dependency and the only third-party runtime
dependency in the repository — it is used solely by the `heerwisch-jfreechart`
driver module. It is linked dynamically as a Maven artifact, so downstream
consumers retain their LGPL right to replace it without having to re-license
this project. Consumers that do not need chart rendering can depend on the
other modules and never pull it in.
